# Шестиугольный Numberlink (Hex Numberlink Solver)

Проект представляет собой игру и автоматический решатель головоломки **Numberlink** на гексагональной (шестиугольной) сетке с графическим интерфейсом на базе библиотеки **Pygame**.

Алгоритм использует поиск с возвратом (**Backtracking**), находит все возможные уникальные пути для соединения пар цифр и отображает пошаговые решения на экране.

---

## Структура проекта

* `numberlink.py` — основной скрипт: парсер файлов карт, логика гексагональной сетки, алгоритм поиска решений и GUI-визуализатор.
* `tests.py` — модульные тесты (`unittest`) для проверки корректности парсера, геометрии соседей и количества решений.
* `input.txt` — файл с входными данными (размер поля и координаты стартовых цифр).
* `.gitignore` — файл, скрывающий технический кэш Python (`__pycache__`) и временные файлы от Git.

---

## Формат входных данных (`input.txt`)

Карта описывается в текстовом файле по следующим правилам:
1. Строка `SIZE X Y` задаёт максимальные индексы поля по горизонтали и вертикали.
2. Строки формата `X Y N` задают координаты ячейки (`X`, `Y`) и расположенную в ней цифру (`N`).
3. Символ `#` используется для комментариев (такие строки полностью игнорируются парсером).

**Пример корректного файла:**
```text
SIZE 4 4
# Пары единиц
0 0 1
3 3 1
# Пары двоек
1 1 2
2 2 2
